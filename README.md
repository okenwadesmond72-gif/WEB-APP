# Professional Task Management & Productivity Web App

A modern, full-stack web application for managing tasks, projects, and team collaboration.

## Features

- User Authentication (JWT)
- Create, Read, Update, Delete Tasks
- Project Management
- Task Categories & Priority Levels
- Due Dates & Reminders
- Team Collaboration
- Responsive UI (Mobile & Desktop)

## Tech Stack

### Frontend
- React 18+
- TypeScript
- Tailwind CSS
- Redux Toolkit
- Axios
- React Router

### Backend
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication
- Bcrypt
- CORS

## Installation

1. Clone the repository
2. Install dependencies (cd client && npm install, cd ../server && npm install)
3. Configure .env file
4. Run: npm run dev (backend) and npm start (frontend)

## Access

- Frontend: http://localhost:3000
- Backend API: http://localhost:5000
